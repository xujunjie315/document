## 1、http协议基础及发展历史

### 1.1 网络协议分层

```bash
    应用层：http,ftp
    传输层:向用户提供可靠的端到端服务，tcp,udp
    网络层：数据在节点之间传输船舰逻辑链路
    数据链路层：在通信的实体间建立数据链路连接
    物理层:定义物理设备如何传输数据
```
